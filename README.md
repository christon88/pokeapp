## Setup

Note: The setup assumes you have brew installed and are working on MacOS (or UNIX compatible OS(?))

1 Clone repo
```
git clone https://github.com/christon88/pokeapp.git
cd pokeapp
```

2 Install node, yarn and dependencies
```
brew install node
brew install yarn
yarn

```

3 Run app
```
yarn start
```
